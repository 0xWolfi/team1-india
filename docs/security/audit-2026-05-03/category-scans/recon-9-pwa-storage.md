## 5. Background Sync (lib/backgroundSync.ts)

### 5.1 Sync Queue & Retry Logic

**Mechanism:** Client-side fetch queue (NOT Web Background Sync API).  
**Storage:** IndexedDB `pendingActions` store (shared with offline forms).

**Flow:**
1. Request enqueued via `queueRequest(url, options)` (line 15)
2. Auto-inferred type from URL (line 181‚Äì187): application, vote, comment, experiment, contribution
3. Attempt immediate sync if `networkMonitor.isStableOnline()` (line 28)
4. `processQueue()` iterates all pending actions (line 56)

**Per-action retry logic** (line 78‚Äì130):
- Max retries: 3
- Backoff: exponential (1s, 2s, 4s)
- On 409 (Conflict): assume duplicate, delete without retry
- On 4xx/5xx: retry with backoff
- On network error: retry with backoff

**Endpoint called:** Stored in `action.endpoint`; re-issued with original headers/body.  
**Re-authentication:** ‚ùå NO ‚Äî headers re-sent as-is (likely expired tokens if offline > TTL).  
**Risk:** If user goes offline, access token expires, sync will fail with 401 but keep retrying.

**Idempotency:** Deduplication via `idempotencyKey = hash(type + body)` (offlineStorage.ts:269‚Äì286).  
‚Üí Prevents duplicate submissions if same action queued twice.

| Risk | Severity | Mitigation |
|------|----------|-----------|

| Device ID in plaintext localStorage | Medium | Enables encryption key recovery if localStorage compromised |

### ‚ùå GAPS & IMPROVEMENTS

1. **Add manifest.json** ‚Äî declare app scope, icons, start_url
2. **Validate push notification URLs** ‚Äî origin check before `clients.openWindow()`
3. **Encrypt IndexedDB pending actions** ‚Äî wrap with Dexie encryption plugin or server-side commit
4. **Reduce core-pages TTL** ‚Äî 5 min ‚Üí 1 min to avoid stale dashboard
5. **Sign push subscriptions** ‚Äî include HMAC of endpoint in DB to prevent hijack
6. **Implement Web Background Sync API** ‚Äî offload queue retry logic to browser (when online, automatically retry pending)
7. **Rotate PBKDF2 salt** ‚Äî use per-user salt or per-session salt instead of static
